# Smart File Organizer 🧠📂

A Python automation tool that organizes files into categories like Images, Videos, Documents, etc.

## Features
- Automatically sorts files into folders
- Supports multiple file formats
- Simple and fast

## How to Use
1. Run the script:

python organizer.py

2. Enter the folder path
3. Done 🎉

## Tech Used
- Python
- OS module
- Shutil module
